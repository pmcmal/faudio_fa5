TF升级方法：
拷贝固件拷贝内存卡的根目录，然后在系统设置-固件升级操作；
升级注意事项：
1. 确保电量30%以上；（电量不足会提示，不允许升级）
2. 确保下载的固件完整；
---------------------------------------------------------------------------
PC端升级工具说明；
1. 将播放器关机；
2. 按住音量减不放连接电脑；
3. 打开工具包，点Update升级；
升级注意事项：
1. 台式电脑建议接机箱后面的USB接口，以确保连接稳定（不要接HUB或者集成线上面）；
2. 确保下载的固件完整；
3. 如果升级失败没有关系，长按开机键10秒，再重复升级步骤一次；
